# Roadmap

## Phase 0: Shape the MVP

- choose first MVP path: personal website, productivity app, or both
- choose 2-3 initial tool routes
- define exact free tier
- define exact paid upgrade
- finish first 30 course blocks
- design idea intake and archetype matcher

## Phase 1: MVP

- create coding.kitty/learn entry point
- build personal website learning path
- generate course map from intake
- add progress tracking
- add prompt templates
- add version control checkpoints
- add deployment walkthrough
- collect learner feedback

## Phase 2: Paid Path

- build simple productivity app path
- add deeper Git/GitHub module
- add infrastructure/deployment module
- add templates
- add project review or community support
- test paid conversion from the free path

## Phase 3: Platform Expansion

- more project tracks
- project showcase
- sponsor-backed labs
- challenges
- cohort offering
- certificates or badges
