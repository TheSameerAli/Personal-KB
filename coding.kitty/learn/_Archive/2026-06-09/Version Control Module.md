# Version Control Module

## Learning Goal

Learners should understand version control as a practical safety net for AI-assisted building.

## Topics

- what version control is
- why version control matters when AI edits code
- Git vs GitHub
- repositories
- commits
- branches
- pull requests at a beginner level
- reading diffs
- recovering from mistakes
- publishing a project on GitHub
- asking an AI agent to explain code changes before committing

## Teaching Angle

Version control should not be framed as developer ceremony. It should be framed as:

> Save points for your project.

Learners should build the habit of saving meaningful checkpoints after each successful task.
