# Monetization Strategy

## Freemium Model

### Free Tier

- intro lessons
- basic idea intake
- one beginner project archetype path
- basic prompt templates
- basic progress tracking
- community access, if desired

### Paid Tier

- full project paths
- deeper personalized course maps
- advanced project archetypes
- productivity app path
- automation workflow modules
- AI agent playbooks
- templates
- project reviews
- office hours
- certificates or badges
- challenges and hackathons

## Sponsor-Supported Revenue

- tool-specific modules
- sponsored build challenges
- "build with X" labs
- partner resource pages
- hackathon tie-ins
- education campaigns for developer tools

## Pricing Ideas

- one-time paid project path
- monthly membership
- annual membership
- cohort-based premium experience
- team or school licenses
