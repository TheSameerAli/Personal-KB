# MVP Scope

## MVP Learning Path

> Build and deploy your first personal website with AI.

## MVP Goal

Prove that a non-technical learner can use a guided workspace to build, understand, version, and deploy a simple real project with AI assistance.

## Included

- Coding Kitty Studio identity
- structured lesson path
- interactive task checklist
- prompt templates
- tool-agnostic guidance
- support for 2-3 beginner-friendly tools
- basic software fundamentals
- beginner-friendly version control flow
- deployment walkthrough
- simple infrastructure explainer
- completion page or project showcase

## Not Included Yet

- full in-browser IDE
- custom AI agent execution
- complex backend projects
- paid cohort operations
- advanced certificates
- large marketplace of templates

## MVP Questions

- Should the first version guide users through external tools, or include a lightweight built-in workspace?
- Which tools should be supported first?
- Should project sharing/showcase be included in MVP?
- What should be free versus paid?
