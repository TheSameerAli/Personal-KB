# Product Vision

## Vision

Help non-technical people become capable AI-assisted software builders.

## Core Promise

Go from idea to deployed app using AI coding agents, while learning the software basics needed to stay in control.

## Positioning

Coding Kitty Studio should feel like a guided workspace for turning ideas into shipped software, not a traditional coding course.

## What Makes It Different

- project-first learning
- AI-guided development
- practical software fundamentals
- version control as a safety habit
- infrastructure and deployment explained through real projects
- friendly Coding Kitty brand layer without making the product feel unserious

## Product Principles

- Build first, explain when needed.
- Teach learners how to guide AI, not blindly trust it.
- Make every concept connect to a real shipped project.
- Treat version control and deployment as core skills.
- Keep the experience friendly, practical, and confidence-building.
