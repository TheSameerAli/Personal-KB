# coding.kitty — Brand Prospect Research

## Purpose

Curated shortlist of **high-quality brands and products** that fit the `coding.kitty` brand, current video topics, and sponsor shape.

This is intentionally **not** a giant sloppy lead dump. I filtered for brands that are:

- genuinely developer-facing
- strong enough brands to help `coding.kitty` level up credibility
- aligned with existing content themes like AI tooling, cloud, infra, APIs, DevOps, security, and beginner-friendly coding explainers
- backed by **official references** from first-party sites
- paired with **publicly listed contact emails** from official pages wherever possible

## Existing context

Known from earlier notes and active sponsorship work:

- `coding.kitty` already has visible sponsor/partner signals with [[Temporal]] and [[Aikido Security]]
- `coding.kitty` / Sameer has also already **worked with Microsoft**
- User update on 2026-05-25: outreach / work has already been done with **JetBrains**, **Microsoft**, and **GitHub Copilot**
- that means the best next step is not random outreach — it is to expand into **adjacent high-trust developer brands** and deprioritize already-worked brands in the next outbound wave

## Selection logic

I prioritized brands that fit one or more of these content lanes:

1. **AI for developers**
2. **cloud / platform / infra explainers**
3. **DevOps / CI/CD / workflow / platform engineering**
4. **security / secure coding / agent governance**
5. **APIs / developer productivity / coding tools**
